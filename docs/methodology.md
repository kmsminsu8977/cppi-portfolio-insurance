# 방법론 노트

## 1. 프로젝트 목적

고정 floor와 multiplier를 사용해 위험자산 노출이 어떻게 동적으로 변하는지 계산한다.

## 2. 핵심 질문

초기 보호수준을 유지하면서 상승장 참여도를 높이려면 multiplier가 어떤 역할을 하는가?

## 3. 입력 데이터

- 파일: `data/sample/strategy_returns.csv`
- 성격: 강의자료나 외부 원천을 복제하지 않은 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. cushion은 portfolio value와 고정 floor의 차이로 정의한다.
2. 위험자산 비중은 multiplier x cushion/value로 계산하고 0~100%로 제한한다.
3. 시장 경로는 합성 수익률이며 실제 상품 약관이나 운용성과를 복제하지 않는다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
